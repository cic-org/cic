## Installation
### Client Node
``` bash
git clone https://github.com/cic-org
cd cic-org
./client -datadir CIC -port 9006 -peer http://*.*.*.*:* -model 2
```
### Main Node
``` bash
git clone https://github.com/cic-org
cd cic-org
./core -datadir TEST -port 9999
```
